My Projects

Task-Tracking App - Designed and developed a customizable task/expense tracking progressive web application using Vue.js framework

Medical Services Recommender App - Developed and deployed web application using Django framework that recommends health services for a queried symptom based on geographic location using Yelp reviews
Implemented recommendation system using NLP/IR techniques such as cosine similarity and LDA topic modeling to classify and retrieve relevant reviews.

Mood Over Time: How Artist’s Music Changes During their Careers - Characterizing a music artist’s evolution using sentiment analysis and KL Divergence.

Which phone does President Trump use to tweet from? - The goal is to classify the whether Donald Trump used an android or iphone to tweet from.

Classifying the likelihood of divorce - Using a publicly available divorce dataset from UCI, I wanted to determine whether one could predict whether an individual is likely to divorce. The divorce data looks at 170 individuals (84 divorced and 86 married) who answered 54 questions or predictors on the Divorce Predictor Scale. Each question is on a five-point scale from 0 (strongly agree) to 4 (strongly disagree).

How well do kinematics predict homerun trajectories? - Utilized the MLB Statcast data for hardest-hit baseballs based on Exit Velocity from 2015-2017 to compare actual homerun distances to predicted homerun distances based on a simple physics model
